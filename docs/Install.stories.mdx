# Install

1. Node package를 설치합니다.

   ```shell script
   yarn add @didi-santa/didicast-modules
   ```

2. 최상위 Sass 파일, 또는 가능한 상위 컴포넌트(일반적으로 router가 위치한 컴포넌트)에 스타일을 설치합니다. (Optional)

   ```javascript
   import "@didi-santa/didicast-modules/lib/styles/commons.scss";
   ```

   Or

   ```scss
   @import "~@didi-santa/didicast-modules/lib/styles/commons.scss";
   ```

3. 원하는 모듈을 사용합니다.
